Keywords
FN		Function

ELSE	ElseIf or Else
OPUT	Print
LOOP	Loop (For or While)
OPEN	Open File
RE		Regular Expression
CLS		Class
#LN		Length

			d=Day(1-31) 		/ m=Month(1-12)		/ y=Year(xx)
			D=Weekday(1-7) 		/ M=Month(Jan-Dec)	/ Y=Year(xxxx)
			w=Week(1-52)		/ W=Weekday(Mo-Su)
			h=Hour(1-12am/pm)	/ H=Hour(1-24)
			n=Minutes(0-59)		/ s=Seconds(0-59)
			l=LeapYear(True or False)
#LF		Left
#RG		Right
#MD		Mid
STR 	String (Value,param1,param2)
NUM		Number (Value,param1)

			0 = Integer 
			1+ = Float 
			Negative Numbers = Absolute Floats
#IT		Integer
#FL		Float
#ST		String
RTN		Return

CASE 	String Case (Value,Param1)
IPUT	Input
IMP		Import modules
TYPE	Type of Object
